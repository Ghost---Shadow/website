# Fractional Gamma Function via Fractional Derivatives

<date>18 Sep 2025</date>

## Introduction

The classical gamma function <InlineMath math="\Gamma(z)"/> is one of the most important special functions in mathematics, serving as a continuous extension of the factorial function. It satisfies the fundamental recurrence relation <InlineMath math="\Gamma(z+1) = z \cdot \Gamma(z)"/>, which represents a "unit step" in the argument.

But what if we could generalize this to fractional steps? What if instead of moving from <InlineMath math="z"/> to <InlineMath math="z+1"/>, we could move in increments of <InlineMath math="\alpha"/>, where <InlineMath math="\alpha"/> is any positive real number? This question leads us into the fascinating world of fractional calculus and opens up new mathematical structures with applications ranging from fractional differential equations to mathematical physics.

In this post, we'll explore how fractional derivatives naturally give rise to a generalized gamma function with fractional step sizes. This isn't just a mathematical curiosity â€“ these functions appear naturally in solutions to fractional differential equations and systems with memory effects or fractional dimensionality.

The key insight is that just as the classical gamma function emerges from integer-order derivatives of <InlineMath math="x^n e^{-x}"/>, we can use fractional derivatives to create a new family of functions that satisfy fractional recurrence relations of the form <InlineMath math="\Gamma_{\alpha}(z+\alpha) = z \cdot \Gamma_{\alpha}(z)"/>.

## Classical Setup

Starting with the function family:
<BlockMath math="g_n(x) = x^n e^{-x}"/>

The classical gamma function arises from:
<BlockMath math="\Gamma(n+1) = \int_0^{\infty} x^n e^{-x} \, dx = \int_0^{\infty} g_n(x) \, dx"/>

With the recurrence relation:
<BlockMath math="\Gamma(n+1) = n \cdot \Gamma(n)"/>

## Fractional Derivative Foundation

### Riemann-Liouville Fractional Derivative

For a function <InlineMath math="f(x)"/> and fractional order <InlineMath math="\alpha > 0"/>, the Riemann-Liouville fractional derivative is:

<BlockMath math="D^{\alpha} f(x) = \frac{1}{\Gamma(n-\alpha)} \frac{d^n}{dx^n} \int_0^x (x-t)^{n-\alpha-1} f(t) \, dt"/>

where <InlineMath math="n = \lceil \alpha \rceil"/>.

### Fractional Derivative of Power Functions

For <InlineMath math="f(x) = x^{\beta}"/> with <InlineMath math="\beta > -1"/>:
<BlockMath math="D^{\alpha} x^{\beta} = \frac{\Gamma(\beta+1)}{\Gamma(\beta+1-\alpha)} x^{\beta-\alpha}"/>

## Fractional Extension of $g_n(x)$

### Step 1: Define Fractional Order Functions

For any real <InlineMath math="\nu \geq 0"/>, define:
<BlockMath math="g_{\nu}(x) = x^{\nu} e^{-x}"/>

### Step 2: Fractional Derivative Relation

The fractional derivative of order <InlineMath math="\alpha"/> gives us:
<BlockMath math="D^{\alpha} g_{\nu}(x) = D^{\alpha}[x^{\nu} e^{-x}]"/>

Using the fractional product rule (generalized Leibniz rule):
<BlockMath math="D^{\alpha}[x^{\nu} e^{-x}] = \sum_{k=0}^{\infty} \binom{\alpha}{k} D^k[x^{\nu}] \cdot D^{\alpha-k}[e^{-x}]"/>

For <InlineMath math="\nu > \alpha"/>, the fractional derivative gives:
<BlockMath math="D^{\alpha} g_{\nu}(x) = \frac{\Gamma(\nu+1)}{\Gamma(\nu+1-\alpha)} x^{\nu-\alpha} e^{-x} = \frac{\Gamma(\nu+1)}{\Gamma(\nu+1-\alpha)} g_{\nu-\alpha}(x)"/>

## Fractional Gamma Function Definition

### Definition 1: Direct Fractional Integration

<BlockMath math="\Gamma_{\alpha}(z) = \int_0^{\infty} x^{z-1} e^{-x^{\alpha}} \, dx"/>

This reduces to the classical gamma function when <InlineMath math="\alpha = 1"/>.

### Definition 2: Fractional Step Recurrence

Define the fractional gamma function via:
<BlockMath math="\Gamma_{\alpha}(z+\alpha) = \frac{\Gamma(z+\alpha)}{\Gamma(z)} \cdot \Gamma_{\alpha}(z)"/>

More explicitly:
<BlockMath math="\Gamma_{\alpha}(z) = \frac{\Gamma(z)}{\Gamma(z-\alpha)} \quad \text{for } z > \alpha"/>

### Definition 3: Integral Transform Approach

<BlockMath math="\Gamma_{\alpha}(z) = \frac{1}{\alpha} \int_0^{\infty} t^{z/\alpha - 1} e^{-t} \, dt"/>

## Properties of the Fractional Gamma Function

### Fractional Recurrence Relation

<BlockMath math="\Gamma_{\alpha}(z+\alpha) = z \cdot \Gamma_{\alpha}(z)"/>

This is the key property - instead of unit steps, we have fractional steps of size <InlineMath math="\alpha"/>.

### Relationship to Classical Gamma

<BlockMath math="\Gamma_1(z) = \Gamma(z)"/>

<BlockMath math="\Gamma_{\alpha}(z) = \frac{\Gamma(z)}{\Gamma(z-\alpha)} \quad \text{when well-defined}"/>

### Functional Equation

For positive integer <InlineMath math="n"/>:
<BlockMath math="\Gamma_{\alpha}(z+n\alpha) = z(z+\alpha)(z+2\alpha)\cdots(z+(n-1)\alpha) \cdot \Gamma_{\alpha}(z)"/>

## Special Values

### At Integer Multiples
For integer <InlineMath math="n \geq 1"/>:
<BlockMath math="\Gamma_{\alpha}(n\alpha) = \frac{\Gamma(n\alpha)}{\Gamma((n-1)\alpha)} \cdot \frac{\Gamma((n-1)\alpha)}{\Gamma((n-2)\alpha)} \cdots \frac{\Gamma(\alpha)}{\Gamma(0)}"/>

This telescopes to:
<BlockMath math="\Gamma_{\alpha}(n\alpha) = \frac{\Gamma(n\alpha)}{\Gamma(0)}"/>

However, since <InlineMath math="\Gamma(0)"/> is undefined, we use the recurrence relation instead.

### Half-Integer Case (<InlineMath math="\alpha = 1/2"/>)
<BlockMath math="\Gamma_{1/2}(z) = \frac{\Gamma(z)}{\Gamma(z-1/2)}"/>

For <InlineMath math="z = 3/2"/>:
<BlockMath math="\Gamma_{1/2}(3/2) = \frac{\Gamma(3/2)}{\Gamma(1)} = \frac{\sqrt{\pi}/2}{1} = \frac{\sqrt{\pi}}{2}"/>

## Applications

This fractional gamma function naturally appears in:

1. **Fractional Differential Equations**: Solutions to <InlineMath math="D^{\alpha} u + \lambda u = 0"/>
2. **Fractional Probability Distributions**: Generalizations of gamma and beta distributions  
3. **Special Functions**: Connections to Mittag-Leffler functions and Fox H-functions
4. **Mathematical Physics**: Systems with fractional dimensionality or memory effects
